🍽️ Restaurant Management System
A terminal-based Restaurant Management System built using Python and SQLite. This project was developed to manage day-to-day restaurant operations through a simple command-line interface.
It features a secure login system with role-based access (Admin & Staff), where passwords are stored using SHA-256 hashing. The system handles everything from billing and employee records to dish management and revenue tracking.
Key highlights:

🔐 Login system with hashed passwords and max attempt lockout
🧾 Create, search, and view customer bills
🍛 Manage dishes and pricing
👨‍💼 Add, remove, and update employee records
💰 View daily and date-range revenue collections
📊 Analytics — top customers, popular dishes, business summary
📁 Export data to CSV (Admin only)
👥 User management — add/remove staff accounts (Admin only)

Built entirely with Python's standard library — no external packages required. Just clone, run main.py once to set up the database, and start the app with restaurant_with_login.py.
